# Yunth 美容液診断ページ

Yunthの4種類の美容液から、肌悩みに合った1本を提案する分岐型の診断ページです。単体で完結するHTMLファイル（`index.html`）です。

## 特徴

- フルスクリーンのピンク系グラデーション背景 / ガラス風カードデザイン
- メインコンテンツ最大幅 900px、PC・スマートフォン対応
- 分岐型診断（最短1問・最大2問）で必ず4商品のいずれかを提案
- 質問切替・結果表示のフェード/スライドアニメーション
- 診断結果ページ、4商品一覧、注釈・注意事項を掲載

## 診断フロー

- **Q1**「シミ・透明感をケアしたい」→ 生VC美白美容液
- **Q1**「年齢サインや肌質を整えたい」→ Q2 へ
  - **Q2**「シワやハリ不足」→ 生レチノール美容液
  - **Q2**「皮脂・テカリ・毛穴」→ 生アゼライン酸美容液
  - **Q2**「ごわつき・キメの乱れ・ツヤ不足」→ 生PDRN美容液

## 使い方

`index.html` をブラウザで開くだけで動作します。

## 差し替え箇所

- ファーストビュー: `.fv` 内にコメントで `<picture>`（PC 1600×700〜800 / SP 750×1000〜1200）構造を用意
- 商品画像: `.result__visual` / `.product-card__thumb` を `<img>` に置換可能
